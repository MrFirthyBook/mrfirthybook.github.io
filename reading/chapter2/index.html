			Reading List: Chapter 2: Blindness

For easy access, here are the links to additional resources discussed in Chapter two of 'Practical Web Accessibility'. These links will provide supplementary information and learning from what's covered in the chapter, allowing you to deepen your understanding in this area.

Reading List

Practical example — ARIA & Headings

A series of examples using ARIA attributes, and accessible slider component, and an overview of how heading hierarchies should work.

Practical example — Skip to main content

Hit the tab key to bring up the 'Skip to main content' link and hit Enter to be sent directly past the common page items such as the menu to the unique page content.

WAI-ARIA Roles

A full list of the ARIA roles, states, and properties available to be added to websites.

ARIA Authoring Practices Guide

A guide to implementing ARIA correctly. I highly suggest looking through it to explore all the possible additions you can make, as you make new features and experiences.

W3C - ARIA Patterns

Learn how to make accessible web components and widgets with ARIA roles, states and properties and by implementing keyboard support.

Sara Soueidan - Screen reader setup guide

An incredibly thorough setup guide for installing and setting up screen reader software on a range of platforms and devices.

Example Source Code - Practical Web Accessibility

The full code repository for the book, featuring every example in every chapter.
