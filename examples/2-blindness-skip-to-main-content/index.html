			Today's lesson: 'Skip to main content' link

How to use the link

Hit the tab key to bring up the 'Skip to main content' link and hit Enter to be sent directly past the common page items such as the menu to the unique page content.

Testing the link

Try using this page with your screen reader of choice, and experience the ease of use and time saved thanks to the link. To further show the impact of this, in this project folder you will also find a file called index-without-link.html. This is, unsurprisngly, a replica of this page but without the 'Skip to main content' link added. You should try using that page with a screen reader too, and see how frustating not having this feature can be.
